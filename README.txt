This is Employee Attendance System For Rift Valley University, which enable the university to take daily employee attendance

Some of Functionalities For Admins are:

	Login to the system.
	Register Employee and Store to database  with thier Employee Number.
	Edit and Delete Employee by any means.
	Register Users Who are admins with username and password and their full name and store to database.
	Edit and Delete user admins by any means.
	View Employee List who register Before on The system.
	View UserList or Admins who already registered on the system.
	View daily Employee Attendance List
	filter each user Attendance by their name and can make summary from the data.

Functionalities for Employee:
	Fill the attendance by using his/her Employee Number.

the attendance filing time is restricted for given time,
                2:00-2:30 LT
		5:00-6:00 LT
		8:00-8:30 LT
		11:00-11:30 LT
It can be changed depending on the university working time schedule.
